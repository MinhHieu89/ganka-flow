Modal Trigger Rules

  • Modal A — ⋮ menu → "Check-in" on a "Chưa đến" row with complete data
  • Modal B — same trigger, but system detects missing required fields (DOB, gender, reason)
  • Modal C — search bar → select existing patient → opens walk-in modal
  • "Sửa thông tin" link navigates to /intake/:id/edit
  • "Check-in & bổ sung hồ sơ →" navigates to /intake/:id/edit, auto-checks-in on save
