Form Behavior

  • Patient ID — auto-generated GK-YYYY-NNNN, shown but not editable
  • 5 required fields — Họ tên, Giới tính, Ngày sinh, SĐT, Lý do khám (red asterisk)
  • Duplicate phone — yellow warning + "Mở hồ sơ cũ" link appears real-time on phone input
  • Save actions — "Lưu" returns to dashboard, "Lưu & chuyển Sàng lọc" transitions to Pre-Exam
  • Edit mode (/intake/:id/edit) — same form, pre-filled with existing data
  • Validation — inline errors below fields, red border on invalid fields
