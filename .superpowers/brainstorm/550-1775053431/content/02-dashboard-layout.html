Dashboard Components

  • Action bar — "Tiếp nhận BN mới" (→ /intake/new), "Đặt lịch hẹn" (→ /schedule/new), Search box
  • 5 KPI cards — Lịch hẹn, Chờ khám, Đang sàng lọc, Đang khám, Hoàn thành
  • Status filter tabs — Click to filter table by status (+ "Tất cả")
  • Patient table — 8 columns: STT, Họ tên, Năm sinh, Giờ hẹn, Nguồn, Lý do khám, Trạng thái, Thao tác
  • "Chưa đến" rows — Subtle gray background, italic reason text, "Check-in" button
  • Other rows — ⋯ menu (Xem hồ sơ, Sửa thông tin, Hủy lượt khám)
  • Pagination — 10/20/50 rows per page
